What's new in CSS3

CSS3 is a new CSS standard version. Many parts of it are still in progress and browsers are gradually adding new specifications to their engines.
CSS3 offers new design capabilities which greatly simplify common design tasks.

Browser support

Browser are gradually support new CSS standard, but since it's not yet officially released browser prefer to use custom property names, for example:
-moz-border-radius instead of just border-radius
The consequence of this is that you have to provide different property for each browser

Shadows

CSS3 introduces support for shadows. Shadows are very popular design choice and usually were done by placing shadow images under container for block elements and using images instead of text for text shadows.
Luckily CSS3 solves both of these problems:

Box shadow

The syntax is following: 

.shadow
{
    -moz-box-shadow: 3px 3px 5px 2px #bbb;
    -webkit-box-shadow: 3px 3px 5px 2px #bbb;
    box-shadow: 3px 3px 5px 2px #bbb;
}

Let's discuss property values in order of appearance:

  1. The horizontal offset of the shadow, positive means the shadow will be on the right of the box, a negative offset will put the shadow on the left of the box.
  2. The vertical offset of the shadow, a negative one means the box-shadow will be above the box, a positive one means the shadow will be below the box.
  3. The blur radius (optional), if set to 0 the shadow will be sharp, the higher the number, the more blurred it will be.
  4. The spread radiusÂ (optional), positive values increase the size of the shadow, negative values decrease the size. Default is 0 (the shadow is same size as blur).
  5. Color
Shadowed div

    
.shadow {
  -moz-box-shadow: 3px 3px 5px 2px #bbb;
  -webkit-box-shadow: 3px 3px 5px 2px #bbb;
  box-shadow: 3px 3px 5px 2px #bbb;
}
#shadowedDiv {
  width: 100px; 
  height: 100px;
  margin: 50px;   
}

Text shadow

Another beautiful addition to CSS standard is text shadow. It allows to achieve really nice effects with simple means.
Syntax is following: 

#myDiv {
    text-shadow: 1px 2px 3px #666;
}

Values are:

  1. Distance of the shadow in horizontal direction(x)
  2. Distance in vertical direction(y)
  3. Blur of the shadow
  4. Colour of the shadow

Please notice that text-shadow is already widely supported, so there are not any browser-specific properties

Shadowed text

    
#textShadow {
    text-shadow: 1px 2px 3px #666;
    font-size: 24px;
}

Gradients

Support for gradients is one of the most inconsistent across the browsers.
Look at the syntax to display gradient from black to white in popular browsers: 

.gradient {
    W3C Syntax:
    background: linear-gradient(left top,white, black);
    WebKit Syntax:
    background: -webkit-gradient(linear, left bottom, right top, from(#fff), to(#000));
    Mozilla Syntax:
    background: -moz-linear-gradient(left,  #fff,  #000);
    Microsoft Syntax:
    filter: progid:DXImageTransform.Microsoft.gradient(startColorstr='#ffffff', endColorstr='#000000');
}

You have to test especially gradients (as any other css3 properties) in different browsers because of very different gradient implementations. 


    
.gradient {
    background: linear-gradient(left top,white, black);
    background: -webkit-gradient(linear, left bottom, right top, from(#fff), to(#000));
    background: -moz-linear-gradient(left,  #fff,  #000);
    filter: progid:DXImageTransform.Microsoft.gradient(startColorstr='#ffffff', endColorstr='#000000');
}
#gradientDiv {
  width: 100px; 
  height: 100px;
  margin: 50px;   
}

Border radius

Rounded corners have become insanely popular in web design in recent years. To make corners round before css3 designer would have to use images in every corner of the element.
Disadvantages are clear:
1. Too much markup. 2. Images had to be adjusted for background color of the element, so if color would change - images had to be redone accordingly.

Browser support is similar to the one of border-shadow property - each popular browser has it's own prefix.
Here is example syntax: 

    border-radius: 10px; /* all corners with the same radius */
    border-top-left-radius: 10px; /* only upper left corner will be rounded */
    /* Webkit and Mozilla */
    -moz-border-radius: 10px;
    -webkit-border-radius: 10px;

Usually you have to put all three variations in your css so major browsers are supported. 

           


        
.rounded {
    -webkit-border-radius: 10px;
    -moz-border-radius: 10px;
    border-radius: 10px;
}
#roundedDiv {
  width: 100px; 
  height: 100px;
  margin: 50px;   
  border: 2px solid gray;
}
