More about floats

float property is extremely useful while doing css design, but using it has some common pitfalls, which we are going to discuss.

Float property highlights

Quick float highlights:

  • Floated element is displayed as blocked, as if it had display property set to display: block;
  • Floated element will shrink to its content if width is not set specifically
  • Element is "glued" to right or left side of container
  • All other content wraps around floated element

Order of the elements matters

Let's see following use case:
We want to position one block on the left and another on the right. Right block should be glued to the right side and left should expand if container expands.
The naive approach would be: 

Left content that fills all width available and is able to expand when container expands


Right

    
div#floatedRight {
    float: right;
}

As we can see it's not what we wanted - right div is below the left one.
Let's see how browser displays what we wrote:

  1. Render first div - it's block element so expand it to the full width of container
  2. Render second div, move it to the right

It's all simple now: just think where floated element would be displayed normally and move it to the right.
To make our example work we should change the order of divs - first display right floated element and only then left one. 

Right


Left content that fills all width available and is able to expand when container expands

    
div#floatedRight {
    float: right;
}

Another ordering example

Imagine you have three buttons that you want to align against right side of container
So you would write elements one after another and apply float: right; property. 

First


Second


Third

    
div.floatedRight {
    float: right;
}

Order is reversed. Why? Because browser applies styles to elements as they appear in html.

  1. First right floated - move it to the right
  2. Second right floated - move it to the right
  3. Third right floated - move it to the right

Now it's clear why they appear in different order then written in markup.
So all we have to do is to place them in reversed order: 

Third


Second


First

    
div.floatedRight {
    float: right;
}
