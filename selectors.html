Basics. Syntax and Selectors

Type selector

    selector {attribute:  value;}

where the selector is any html tag eg p, h1, body attribute characterize an element such as width, color, background value is the specific value for a attribute, eg '300px', 'yellow', 'green'

Universal selector

    * {attribute:  value;}

This selector selects all elements in html document.

Descendant selector

    descendant1 descendant2 ... descendant {attribute:  value;}

This type of selector lets you select elements that are lower in the hierarchy tree of the document. This enables us for example to give a different font size for links which are placed in paragraph. 

    p a {font-size: 16px;}

Child selector

    parent>  child {feature:  value}

The selector allows you to selects elements that are direct child of a parent. This differs from the previous case, in this case the child is directly inside the parent tag.

Sibling selector

    sibling1 + sibling2 {attribute:  value}

The selector allows you to give this type of attribute elements, which are on the same hierarchical level as the selector before "+" sign, in our case "sibling1".

Attribute selector

    selector[attribute] {attribute: value}

Selector can be any html element such as p, h1, div. An attribute is any attribute. 

    p [title] {font-size: 16px;}

The above example will sets the font size of 16px each p element which has title attribute p [title = "Sample Title"] {font-size: 16px;} The above example, sets the font size of 16px each p element that are given the title attribute with a value of "sample title." This type of selector does not work in IE 6 and older.

Class selectors

    selector.class {attribute:  value}
The selector is any tag p h1, div. Class is a class attribute value (class = "class")

id selector

    selector#id {attribute:  value}

The selector is any tag p, h1, div. id is the id (id = "id")
