Centering things

One of the most common things to do with CSS is centering.

Centering can be vertical or horizontal. While most of the horizontal centering can be solved by using margin: auto; and text-align: center; vertical centering is not so straightforward.

Horizontal centering

Block elements

Block elements can be centered by setting margin-left and margin-right properties to auto.
It will try to make both margins ans big as possible evening them out, so the div will bec centered.

Let's see it on following example:

div#centeredDiv {
    margin: auto;
    border: 2px solid gray;
    width: 100px;
    height: 30px;
}
Centered

Div is not horizontally centered, because right and left margins are the same. Please notice that this doesn't apply to top and bottom margins.

Text

Now let's center that text in the center div.
This is done very simply by using text-align: center;

div#centeredDiv {
    margin: auto;
    border: 2px solid gray;
    width: 100px;
    height: 30px;
    text-align: center;
}
Centered

Vertical centering

Block elements. First approach

One of the ways to center block element vertically requires knowing it's heigh and height of it's parent

div#centeredDiv {
    margin-left: auto;
    margin-right: auto;
    /* 100px/2 - 30px/2 */
    margin-top: 35px;  
    border: 2px solid gray;
    width: 100px;
    height: 30px;
    text-align: center;
}
Centered

Let's see what's going on here:
First we changed our margins to be auto only for left and right properties.
By setting top margin at half of the parent height 100px/2 = 50px we would move top edge of our div to the middle of the parent.
Now we need to move our div to the top so vertical center of parent is the same as vertical center of our div.
We do it by decreasing top margin by half of div's height 30px/2 = 15px, so final margin is 50px - 15px = 35px

Block element. Second approach.

There is another very neat way to center div horizontally and vertically. The major plus of this approach is that there is no need to know dimensions of parent and centered element:

div#centeredDiv {
    position: absolute;
    top: 0;
    bottom: 0;
    left: 0;
    right: 0;
    border: 2px solid gray;
    width: 100px;
    height: 30px;
    text-align: center;
}
Centered

For this approach to work parent element should have position: relative; property set. Absolute position of children will be relative to parent element.
By setting all edges positions to 0 browser will try to stretch inner div t the dimensions to parent, but as child div already has it's width and height it will make it stay in center by "pulling" by all sides with equal "force" :)
Please notice, that if width and height properties wouldn't be set - inner div would stretch to the whole parent content box, which might be often useful.

Text

Centering single line of text is very simple, by setting line-height property to the same as containig div height.

div#centeredDiv {
    margin-left: auto;
    margin-right: auto;
    margin-top: 35px;  
    border: 2px solid gray;
    width: 100px;
    height: 30px;
    text-align: center;
    /* 30px - Same as div's height */
    line-height: 30px;
}
Centered

By setting line-heigh we force font to fill all height, and this will center text vertically

Now let's see how to center multi-line text:

div#centeredDiv {
    border: 2px solid gray;
    width: 100px;
    height: 50px;
    text-align: center;   
    display: table-cell;
    vertical-align: middle;    
}
Centered multiline

To center multiline text we first change it's display property to table-cell which will make browser treat element as table cell.
Table cells in turn can center text vertically, so we use vertical-align: middle;
Major drawback of it is that table cell is not usual block element, so we can't center it by using methods above directly.
To also center containing div we would have to wrap it in block element and center it.
