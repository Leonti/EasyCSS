Easy CSS

Practical test

Your goal here is to make content in left container look the same as image on the right of it.
You just put CSS rules into text area using div ids from html snippet and click 'Apply!' button.
Here is information you will need:

  • Both divs(red and green) have width 200px and height 40px
  • Red divs font is helvetica and greens is times
  • Font sizes are 15px and 14px

There are multiple ways to do it, so just remember what you learned and experiment until you get the same result.
There is example solution provided (one of many ways) by clicking on 'View solution' button.

View solution

div#redDiv,
div#greenDiv {
    width: 200px;
    height: 40px;
    text-align: center;
    line-height: 40px;
}

div#redDiv {
    background-color: red;
    position: absolute;
    top: 0;
    right: 0;
    font-weight: bold;
    font-family: helvetica;
    font-size: 15px;    
}

div#greenDiv {
    background-color: green;
    font-family: times;
    font-size: 14px;
    position: absolute;
    left: 50px;
    top: 130px;
}

Apply!
