Easy CSS

Introduction

This tutorial is aimed at people who just started to style their html markup with CSS but still have some way to go.
Most of the things in this tutorial can be considered basic, but why would you need a tutorial for advanced stuff?
There is a documentation for that :)
Tutorial highlights things that are most commonly used in CSS styling, so it can help you get started.
A lot is shown on practical examples which you can modify and see how it works.
There is also a quizand practical test at the end of the tutorial.
