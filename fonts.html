Fonts

Font sizes

The font size can be set using the attribute font-size:

    selector {font-size:  size}

size can be defined in four different ways:

Name value, absolute:

xx-small
x-small
small
medium
large
x-large
xx-large

Relative value to:

smaller - less than the current
larger - bigger than the current

Unit of length

size 1.2 cm
size 14mm
size 0.45in
size 12pt
size 2pc
size 12px
size 1em
size 2ex
Unit "em" is a relative measure. This refers to the size of the font defined for the parent element. 1em = 100% of the parent

The percentage of current size

size 13%
size 150%

Type of font

    selector {font-family: fonttype1, fonttype2, fonttype3 ...}

The selector is any tag, fonttype... is the name of the font type. If the particular user does not have fonttype1 in his system, webbrowser will render text with fonttype2 and so on. You can also font embedding, eg: 

    @font-face {
        font-family: comic;
        src: local ("Comic Font Regular"),url (http://address/file.ttf) format("truetype");
    }
But this does not work in all popular browsers. If you want to use custom fonts use Google Web Fonts API is definitely the best solution that works in all browsers. Another solution is the Cufon. However, these methods are not the subject of this course.

Font Style

    selector {font-style: style; }

The selector is any tag, style is one of the three values:
normal - normal font
italic - italic font (if unavailable, oblique style is chosen)
oblique - the font italic

The thickness of the font:

    selector {font-weight: weight}

The selector is any tag, weight defines the thickness of the font. Allowed values ​​for weight are: 100, 200, 300, 400, 500 (normal), 600, 700 (bold), 800, 900 ...

Font variant

    selector {font-variant: variant;}

The selector is any tag. The variant options are: normal (normal font) and small-caps (small caps). Caps are capitals, but sized of small letter.
